# Expense Tracker (Трекер расходов)

## Автор
Игорь Леонов

## Краткое описание программы
Приложение для учёта личных расходов. Позволяет:
- Добавлять расходы с указанием суммы, категории, даты и описания
- Просматривать список всех расходов в виде таблицы
- Фильтровать расходы по категории и периоду дат
- Автоматически подсчитывать общую сумму расходов (всех и отфильтрованных)
- Сохранять и загружать данные в JSON

## Как запустить приложение

### Требования
- Python 3.6 или выше

### Запуск
1. Сохраните код в файл `expense_tracker.py`
2. Выполните в терминале:
   ```bash
   python expense_tracker.py
Примеры использования
Добавление расхода:

Сумма: 25.50

Категория: Food

Дата: 2025-05-18

Описание: Обед в кафе

Нажмите "Add Expense"

Фильтрация расходов:

По категории: выберите "Food" → "Apply Filters"

По дате: укажите Start Date и End Date (например, 2025-05-01 до 2025-05-31)

Можно комбинировать оба фильтра

Просмотр статистики:

Общая сумма всех расходов отображается вверху

При применении фильтров показывается сумма только отфильтрованных расходов

Удаление расхода:

Выберите запись в таблице

Нажмите "Delete Selected"

Сохранение/загрузка:

"Save to JSON" → сохраняет в expenses.json

"Load from JSON" → загружает из файла

Структура JSON файла
json
[
    {
        "amount": 25.5,
        "category": "Food",
        "date": "2025-05-18",
        "description": "Обед в кафе"
    }
]
