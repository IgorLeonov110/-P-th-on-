# Random Task Generator (Генератор случайных задач)

## Автор
**Имя и фамилия:** Игорь Леонов

## Краткое описание программы
Приложение для управления списком задач и генерации случайных задач. Позволяет:
- Добавлять задачи с указанием названия, типа и описания
- Просматривать список всех доступных задач
- Фильтровать задачи по типу перед генерацией
- Генерировать случайную задачу из списка (с учётом фильтра)
- Сохранять историю всех сгенерированных задач
- Сохранять и загружать данные в JSON

## Как запустить приложение

### Требования
- Python 3.6 или выше

### Запуск
1. Сохраните код в файл `task_generator.py`
2. Выполните в терминале:
   ```bash
   python task_generator.py
Примеры использования
Добавление задачи:

Название: Сделать отчёт

Тип: Work

Описание: Подготовить квартальный отчёт

Нажмите "Add Task"

Генерация случайной задачи:

Выберите фильтр по типу (или "All")

Нажмите "Generate Random Task"

Приложение покажет выбранную задачу и добавит её в историю

Просмотр истории:

Все сгенерированные задачи отображаются в нижней таблице с датой и временем

Сохранение/загрузка:

"Save to JSON" → сохраняет в task_generator.json

"Load from JSON" → загружает из файла
